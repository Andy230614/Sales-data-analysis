# Sales-data-analysis

# Poject Overview
This project analyzes a retail sales dataset to identify business.

# Tools Used
Python
Pandas
Matplotlib
Power BI(for dashboard)

# Key Questions Answered
- what is the total revenue?
- Which region performs the best?
- Which product categories generate the highest profit?
- What are the top selling products?
- What is the monthly sales trend?

# Key Insights
- The West region generates the highest sales.
- Technology category produces the highest profit.
- Sales trends show seasonal varations.

# Project Structure
Data/ - dataset

Images/ - visualizations
